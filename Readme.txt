Setup Instructions
1. Connect to Cisco secure network using VPN
2. Import the Homework4.war file into Eclipse
3. Start the Tomcat server (The WAR file is compatible with Tomcat version 7)
4. Right click on the project >> Run As >> Run on server
5. In the browser access: http://localhost:<port_no>/Homework4
